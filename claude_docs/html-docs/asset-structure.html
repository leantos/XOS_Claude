import 'bootstrap/dist/css/bootstrap.min.css';  // 1. Bootstrap base

?? Asset Bundling

During build process:

  • SCSS files compile to CSS
  • CSS files are bundled and minified
  • Fonts are copied to build/fonts
  • Images are optimized and included in bundle
  • XOS components are bundled with the application

?? Best Practices

  • XOS Components: Always use from src/xos-components/
  • Custom Styles: Add to src/scss/ and import in theme.scss
  • Module Styles: Keep in module folder as index.css
  • Fonts: Place in src/fonts/ for custom fonts
  • Icons: Use Font Awesome or Material Design Icons
  • Images: Store component images in component folder
  • Build Assets: Never edit files in build/ directory

?? Asset Pipeline

  • Development: Assets served from src/
  • Build Process: SCSS ? CSS, Bundle, Minify
  • Production: Optimized assets in build/ and wwwroot/

This structure ensures consistent asset management across development and production environments.
