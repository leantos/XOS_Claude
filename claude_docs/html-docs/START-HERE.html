?? CRITICAL XOS RULES (Never Break These)

  • Frontend: Always extend XOSComponent, pass VM to super()
  • ViewModel: Always use Data reference pattern: const model = this.Data
  • Event Handlers: Always use 3-step pattern + updateUI()
  • Backend: All endpoints are POST, return domain types directly
  • Database: Always use row.GetValue("column") pattern
  • API Calls: Always use Utils.ajax with relative URLs

Full rules: xos-complete-patterns/ABSOLUTE-RULES.md
